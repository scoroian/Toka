# Homes Feature Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implementar el modelo completo de cuenta de usuario, gestión de hogares (listar, cambiar, abandonar, cerrar) y selector multi-hogar en la cabecera de la app.

**Architecture:** Arquitectura por capas (domain → data → application → presentation) dentro de `lib/features/homes/`. El `HomesRepository` abstracto vive en `domain/`, la implementación Firestore en `data/`, y los providers Riverpod en `application/`. El `CurrentHomeProvider` escucha membresías del usuario en tiempo real y expone el hogar activo a toda la app.

**Tech Stack:** Flutter 3.x, Riverpod (riverpod_annotation), freezed, Cloud Firestore, Cloud Functions (callable), mocktail, fake_cloud_firestore, flutter_test.

```dart
      name: data['name'] as String,
      ownerUid: data['ownerUid'] as String,
      currentPayerUid: data['currentPayerUid'] as String?,

      premiumPlan: data['premiumPlan'] as String?,
      premiumEndsAt: (data['premiumEndsAt'] as Timestamp?)?.toDate(),
      restoreUntil: (data['restoreUntil'] as Timestamp?)?.toDate(),
      autoRenewEnabled: data['autoRenewEnabled'] as bool? ?? false,
      limits: HomeLimits(
        maxMembers: (data['limits']?['maxMembers'] as int?) ?? 5,
      ),
      createdAt: (data['createdAt'] as Timestamp).toDate(),
      updatedAt: (data['updatedAt'] as Timestamp).toDate(),
    );

        orElse: () => BillingState.none,
      ),
      status: MemberStatus.values.firstWhere(
        (e) => e.name == (data['status'] as String? ?? 'active'),
        orElse: () => MemberStatus.active,
      ),
      joinedAt: (data['joinedAt'] as Timestamp).toDate(),
```

```dart
// lib/features/homes/data/homes_repository_impl.dart
import 'package:cloud_firestore/cloud_firestore.dart';
import 'package:cloud_functions/cloud_functions.dart';

import '../../../core/errors/exceptions.dart';
import '../domain/home.dart';
import '../domain/home_membership.dart';
import '../domain/homes_repository.dart';
import 'home_model.dart';

class HomesRepositoryImpl implements HomesRepository {
  HomesRepositoryImpl({
    required FirebaseFirestore firestore,
    required FirebaseFunctions functions,
  })  : _firestore = firestore,
        _functions = functions;

  final FirebaseFirestore _firestore;
  final FirebaseFunctions _functions;

  @override
  Stream<List<HomeMembership>> watchUserMemberships(String uid) {
    return _firestore
        .collection('users')
        .doc(uid)
        .collection('memberships')
        .snapshots()
        .map((snap) =>
            snap.docs.map(HomeModel.membershipFromFirestore).toList());
  }

  @override
  Future<Home> fetchHome(String homeId) async {
    final doc = await _firestore.collection('homes').doc(homeId).get();
    if (!doc.exists) throw ServerException('Home $homeId not found');
    return HomeModel.fromFirestore(doc);
  }

  @override
  Future<String> createHome(String name) async {
    try {
      final callable = _functions.httpsCallable('createHome');
      final result = await callable.call<Map<String, dynamic>>({'name': name});
      return result.data['homeId'] as String;
    } on FirebaseFunctionsException catch (e) {
      if (e.code == 'resource-exhausted') {
        throw const NoAvailableSlotsException();
      }
      rethrow;
    }
  }

  @override
  Future<void> joinHome(String inviteCode) async {
    final callable = _functions.httpsCallable('joinHomeByCode');
    await callable.call<void>({'code': inviteCode});
  }

  @override
  Future<void> leaveHome(String homeId) async {
    final uid = _firestore.app.options.projectId; // placeholder — uid viene del caller
    // Verificar rol antes de salir
    final memberDoc = await _firestore
        .collection('homes')
        .doc(homeId)
        .collection('members')
        .doc(uid)
        .get();

    if (memberDoc.exists) {
      final role = memberDoc.data()?['role'] as String?;
      if (role == 'owner') throw const CannotLeaveAsOwnerException();
    }

    final callable = _functions.httpsCallable('leaveHome');
    await callable.call<void>({'homeId': homeId});
  }

  @override
  Future<void> closeHome(String homeId) async {
    final callable = _functions.httpsCallable('closeHome');
    await callable.call<void>({'homeId': homeId});
  }

  @override
  Future<void> updateLastSelectedHome(String uid, String homeId) async {
    await _firestore.collection('users').doc(uid).update({
      'lastSelectedHomeId': homeId,
    });
  }

  @override
  Future<int> getAvailableSlots(String uid) async {
    final userDoc = await _firestore.collection('users').doc(uid).get();
    if (!userDoc.exists) return 0;
    final data = userDoc.data()!;
    final baseSlots = (data['baseSlots'] as int?) ?? 2;
    final lifetimeUnlocked = (data['lifetimeUnlocked'] as int?) ?? 0;

    final membershipsSnap = await _firestore
        .collection('users')
        .doc(uid)
        .collection('memberships')
        .get();
    final currentCount = membershipsSnap.docs.length;

    return baseSlots + lifetimeUnlocked - currentCount;
  }
}
```

```dart
// lib/features/homes/application/current_home_provider.dart
import 'package:riverpod_annotation/riverpod_annotation.dart';

import '../../auth/application/auth_provider.dart';
import '../domain/home.dart';
import '../domain/homes_repository.dart';
import 'homes_provider.dart';

part 'current_home_provider.g.dart';

@Riverpod(keepAlive: true)
class CurrentHome extends _$CurrentHome {
  @override
  Future<Home?> build() async {
    final auth = ref.watch(authProvider);
    final uid = auth.whenOrNull(authenticated: (u) => u.uid);
    if (uid == null) return null;

    final memberships = await ref.watch(
      userMembershipsProvider(uid).future,
    );
    if (memberships.isEmpty) return null;

    // Leer lastSelectedHomeId del usuario
    final repo = ref.read(homesRepositoryProvider);
    // Intentar cargar el último hogar seleccionado o el primero disponible
    String targetHomeId = memberships.first.homeId;

    try {
      // Obtener lastSelectedHomeId del Firestore (lo leemos vía un campo del user doc)
      // Se pasa por parámetro o se lee del repositorio
      targetHomeId = await _resolveTargetHomeId(uid, memberships, repo);
    } catch (_) {
      // Si falla, usamos el primero
    }

    return repo.fetchHome(targetHomeId);
  }

  Future<String> _resolveTargetHomeId(
    String uid,
    List<HomeMembership> memberships,
    HomesRepository repo,
  ) async {
    // Leer lastSelectedHomeId directo de Firestore
    // Nota: en producción, esto se mueve a un provider dedicado de user profile
    final firestore = (repo as dynamic).firestore as dynamic;
    final userDoc =
        await firestore.collection('users').doc(uid).get() as dynamic;
    final lastId = userDoc.data()?['lastSelectedHomeId'] as String?;

    if (lastId != null &&
        memberships.any((m) => m.homeId == lastId)) {
      return lastId;
    }
    return memberships.first.homeId;
  }

  Future<void> switchHome(String homeId) async {
    final auth = ref.read(authProvider);
    final uid = auth.whenOrNull(authenticated: (u) => u.uid);
    if (uid == null) return;
    await ref
        .read(homesRepositoryProvider)
        .updateLastSelectedHome(uid, homeId);
    ref.invalidateSelf();
  }
}
```

```dart
  final String currentHomeId;

  @override
  Widget build(BuildContext context, WidgetRef ref) {
    final l10n = AppLocalizations.of(context);

    // Obtener uid del auth provider
    final authState = ref.watch(
        // Se importa authProvider para obtener uid
        // ignore: avoid_dynamic_calls
        _authUidProvider(ref));

    if (authState == null) return const SizedBox.shrink();

    final membershipsAsync = ref.watch(userMembershipsProvider(authState));

    return membershipsAsync.when(

  String? _authUidProvider(WidgetRef ref) {
    // Obtenemos uid vía authProvider
    // Importado en el mismo archivo
    return null; // implementado abajo via import correcto
  }

  Widget _buildList(
    BuildContext context,
    WidgetRef ref,
    List<HomeMembership> memberships,
  ) {
    final l10n = AppLocalizations.of(context);

    // Ordenar: 1) último usado (currentHomeId), 2) con tareas pendientes, 3) por nombre
    final sorted = _sortMemberships(memberships);

    return Column(
      mainAxisSize: MainAxisSize.min,
      children: [
        Padding(
          padding: const EdgeInsets.all(16),
          child: Text(
            l10n.homes_selector_title,
            style: Theme.of(context).textTheme.titleMedium,
          ),
        ),
        ListView.builder(
          key: const Key('home_selector_list'),
          shrinkWrap: true,
          itemCount: sorted.length,
          itemBuilder: (ctx, i) => _HomeTile(

            isSelected: sorted[i].homeId == currentHomeId,
            onTap: () {
              ref
                  .read(currentHomeProvider.notifier)
                  .switchHome(sorted[i].homeId);
              Navigator.of(ctx).pop();
            },
          ),
        ),
      ],
    );
  }

  List<HomeMembership> _sortMemberships(List<HomeMembership> memberships) {
    final result = [...memberships];
    result.sort((a, b) {
      // Último usado primero
      if (a.homeId == currentHomeId) return -1;
      if (b.homeId == currentHomeId) return 1;
      // Por nombre
      return a.homeNameSnapshot.compareTo(b.homeNameSnapshot);
    });
    return result;
  }
}
```

```dart
class _HomeSettingsBody extends ConsumerWidget {
  const _HomeSettingsBody({required this.home});

  final Home home;

  @override
  Widget build(BuildContext context, WidgetRef ref) {
    final l10n = AppLocalizations.of(context);
    final uid = ref.watch(authProvider).whenOrNull(authenticated: (u) => u.uid);
    final membershipsAsync =
        uid != null ? ref.watch(userMembershipsProvider(uid)) : null;
    final membership = membershipsAsync?.valueOrNull?.firstWhere(
      (m) => m.homeId == home.id,
      orElse: () => HomeMembership(
        homeId: home.id,
        homeNameSnapshot: home.name,
        role: MemberRole.member,
        billingState: BillingState.none,
        status: MemberStatus.active,
        joinedAt: DateTime.now(),
      ),
    );

    final isOwner = membership?.role == MemberRole.owner;
    final isAdminOrOwner =
        isOwner || membership?.role == MemberRole.admin;
    final isPayer = membership?.billingState == BillingState.currentPayer;

    return ListView(
      key: const Key('home_settings_list'),
      padding: const EdgeInsets.all(16),
      children: [
        // Nombre del hogar
        if (isAdminOrOwner) ...[
          TextField(
            key: const Key('home_name_field'),
            controller: TextEditingController(text: home.name),
            decoration: InputDecoration(
              labelText: l10n.homes_name_label,
            ),
          ),
          const SizedBox(height: 16),
        ] else
          ListTile(
            title: Text(l10n.homes_name_label),
            subtitle: Text(home.name),
          ),

        // Plan
        ListTile(
          key: const Key('home_plan_tile'),
          title: Text(home.premiumStatus == HomePremiumStatus.active
              ? l10n.homes_plan_premium
              : l10n.homes_plan_free),
          subtitle: home.premiumEndsAt != null
              ? Text(l10n.homes_plan_ends(
                  home.premiumEndsAt!.toLocal().toString().split(' ').first))
              : null,
        ),

        // Gestionar suscripción (solo owner/payer)
        if (isOwner || isPayer)
```

```dart
List<HomeMembership> sortMembershipsForSelector(
  List<HomeMembership> memberships, {
  required String currentHomeId,
}) {
  final result = [...memberships];
```

   - Con una cuenta que tiene 2 hogares, abrir la app.
   - Verificar que se abre el último hogar usado.

   - Crear una cuenta nueva → 2 plazas disponibles.
   - Crear hogar 1 → queda 1 plaza.
   - Crear hogar 2 → queda 0 plazas.

   - Ser owner de un hogar.
   - Ajustes → "Abandonar hogar" → debe mostrar snackbar "Transfiere la propiedad antes de abandonar el hogar".

   - Copiar el código.
   - Con otra cuenta en onboarding → "Unirme" → pegar el código → accede al hogar.

6. **Persistencia del hogar activo:**
   - Con 2 hogares, cambiar al hogar 2.
   - Cerrar y reabrir la app.
   - Debe abrirse el hogar 2 (el último seleccionado).
